# Sistema de Gestión de Tickets - SommierCenter

## Descripción
Aplicación web para gestión de tickets de soporte técnico, desarrollada con Flask y SQLite.

## Características Principales
- Creación y edición de tickets
- Seguimiento de estado de tickets
- Gestión de usuarios y colaboradores
- Generación de informes
- Interfaz responsive y moderna

## Tecnologías Utilizadas
- Python
- Flask
- SQLite
- HTML5
- CSS3
- JavaScript

## Requisitos
- Python 3.8+
- Bibliotecas en requirements.txt

## Instalación
1. Clonar repositorio
2. Crear entorno virtual
3. Instalar dependencias
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Ejecución
```bash
python app.py
```
